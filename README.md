# GitHub PR: Lines Viewed Progress Bar

A userscript that adds a progress bar to GitHub Pull Requests, showing the number of lines viewed based on reviewed checkboxes.

## Features

- **Visual Progress Bar**: Adds a progress bar to the review tools header.
- **Line Count**: Shows the exact number of lines viewed vs. total lines changed.
- **Real-time Updates**: Updates immediately when you check/uncheck "Viewed" on a file.
- **Color Coded**: Changes color to green when 100% of lines are viewed.

## Installation

1. Install a userscript manager like [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/).
2. Click [here](https://raw.githubusercontent.com/ilyachch-userscripts/github-pr-lines-viewed-progress-bar/main/github-pr-lines-viewed-progress-bar.user.js) to install the script.

## Usage

1. Open a Pull Request on GitHub.
2. Go to the "Files changed" tab.
3. As you review files, check the "Viewed" checkbox.
4. The progress bar in the header will update to show your progress based on the number of lines in the viewed files.
